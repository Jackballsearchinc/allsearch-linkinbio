# Allsearch Recruiting Links — Static Site (Drag & Drop)

This is a fully static site: just upload these files to GitHub and turn on **GitHub Pages**.

## Publish on GitHub Pages (no build needed)
1) Create a new GitHub repo (e.g., `allsearch-linkinbio-static`).
2) Click **Add file → Upload files** and upload **all files in this folder** (including `index.html`).
3) Commit the upload.
4) Go to **Settings → Pages** → under **Source**, choose **Branch: main** and **/ (root)** → Save.
5) Your site will appear at: `https://<yourusername>.github.io/allsearch-linkinbio-static/`

## Edit Links or Branding
Open `index.html` and look for the `LINKS` array and the `<h1>`/tagline near the top.
